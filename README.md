# Pkpk

The logging pattern based on magic strings for easy detection.

## Installation

Add this line to your application's Gemfile:

    gem 'pkpk'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pkpk

## Usage

```ruby
pkpk []
```
